Atualizando sua app Async-worker
===============================

Essa página descreve os passos necessários para que seja possível você atualizar seu projeto async-worker para uma versão mais atual.

O primeiro ponto de atenção é entender como o async-worker é versionado e isso está descrito com detalhes :ref:`aqui <versioning>`.

Os detalhes de cada release estão no `Github <https://github.com/async-worker/async-worker/releases>`_.

Aqui estão descritos os passos necessários para atualizar para cada um das versões quem trazem mudanças incompatíveis.
