0.13.0¶

Data de release: 2020-03-30

Tag: 0.13.0

  • PR 192 Não remove / à esquerda dos vhosts

Raw Commits: 0.13.0

Notas de atualização¶

Mudança de comportamento no acesso a vhosts RabbitMQ. Antes dessa versão o async-worker fazia lstrip() no nome do vhost de todas as conexões. Isso fazia com que fosse impossível acessar um vhost de nome /foo pois mesmo você passando o nome correto do seu vhost (/foo) o async-worker tentava acessar foo.

Isso significa que: Se você estiver passando o nome do seu vhost para o async-worker com a / inicial você deve ajustar essa passagem para ser sem a /.
